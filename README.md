# 🏨 Hotel Booking System (MERN Stack)

<img width="1891" height="916" alt="image" src="https://github.com/user-attachments/assets/2a1e0b84-0430-4e43-85b2-d58bf31c5ef8" />

This project is a **full-stack Hotel Booking Website** built using the **MERN Stack** — MongoDB, Express.js, React.js, and Node.js. The application allows users to browse available hotel rooms, make bookings, and complete payments online.
The platform also includes an **admin dashboard** where administrators can add new hotel rooms and manage existing bookings.

---

## 🚀 Project Description
The Hotel Booking System provides a complete online reservation experience. Users can create an account, browse available hotel rooms, and book rooms directly from the website. Once a booking is made, users receive an **instant booking confirmation via email**.
The project integrates modern tools and services to handle authentication, payments, and deployment, making it a realistic full-stack application.

---

## ✨ Key Features

### 👤 User Features

* Browse available hotel rooms

### 🛠 Admin Features

* View and manage bookings

## 🧰 Tech Stack

### Frontend
* React.js

### Backend
* Node.js
* Express.js

### Database
* MongoDB

### Authentication
* Clerk

Clerk provides ready-to-use UI components for:
* Sign In
* Sign Up
* User Profile management

### Payment Integration
* Stripe for secure online payments.

### Email Notifications
* Sends booking confirmation emails to users after successful reservations.

### Deployment
* Deployed on Vercel.
